To-Do List Application

A console-based To-Do List application developed in Java that allows users to efficiently manage their tasks. The application provides a simple and interactive interface to handle everyday task management using core Java concepts.

Features:

Add Tasks: Create new tasks with a title, description, due date, priority, and category.
Edit Tasks: Update details of existing tasks.
View Tasks: Display all tasks with their details.
Sort Tasks: Sort tasks by due date, priority, or title.
Filter Tasks: Filter tasks by category, completion status, or overdue status.
Mark Completed: Mark tasks as completed and automatically track overdue tasks.

Technologies Used:

Java: Core programming language.
OOP Concepts: Classes, objects, encapsulation.
Collections: ArrayList for task management.
Date Handling: LocalDate for due dates.
User Input: Scanner for interactive console input.

Why This Project?

This project demonstrates task management functionality and basic CRUD operations in a console environment. It is ideal for learners to understand object-oriented programming, user input handling, and Java collections.
